# M5Stick_Weather + ATOM_LITE 项目开发记录

## 项目概述
使用 M5StickC Plus2（带屏幕）和 M5Atom Lite（无屏幕）采集 SHT30 温湿度传感器数据，
支持 WiFi 热点网页查看、BLE 蓝牙推送、WiFi STA 校园网认证 + HTTP 推送到服务器、
LittleFS 每日归档日志、GPS 坐标显示。

## 关键决策记录

### 1. 传感器调试
- SHT30 (ENV III 模块) I2C 地址 0x44
- Grove 口引脚: M5StickC Plus2 = GPIO32(SDA)/GPIO33(SCL)
- ATOM Lite = GPIO26(SDA)/GPIO32(SCL)
- 原始 I2C 命令绕过 Adafruit_SHT31 库的 CRC 校验问题
- 时钟速度降到 50kHz 解决 Grove 长线稳定性

### 2. BLE 蓝牙
- Nordic UART Service UUID: 6e400001-b5a3-f393-e0a9-e50e24dcca9e
- TX 特征: 6e400002 (notify)
- 手机 nRF Connect 订阅通知接收 JSON 数据

### 3. WiFi 热点
- SSID: M5Stick_Weather / 密码: Dsrdd159987@
- DNS 劫持使手机连接后自动弹出页面
- Web 页面: 实时数据折线图 (纯 Canvas, 无外部依赖)

### 4. 架构重构 (2026-05-11)
- 从单文件 main.cpp (850行) 拆分为 10 个模块
- M5StickC Plus2: config/sht30/rtc/chart/logger/display/ws_server/sta_client/ble
- ATOM Lite: 去掉 display, 用 LED 替代, RTC 改为 uptime
- Windows 文件名大小写问题导致 #include 冲突 → 重命名 webserver → ws_server
